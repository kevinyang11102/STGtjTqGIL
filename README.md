# 电视剧电影推荐系统 python891

本项目是一款基于Python Django框架开发的推荐系统，后端数据库使用MySQL。该系统致力于为用户提供个性化电视剧和电影推荐，通过用户行为分析，实现智能匹配。

## 技术栈

- 后端：Python Django
- 数据库：MySQL
- 前端：HTML, CSS, JavaScript

## 功能模块

- 用户注册与登录
- 电影和电视剧浏览与搜索
- 用户行为记录与分析
- 个性化推荐算法

## 系统角色

- 普通用户：浏览和搜索资源，获取个性化推荐
- 管理员：管理用户信息和资源数据

## 运行环境

- Python 3.8+
- Django 3.2+
- MySQL 5.7+

## 部署步骤

1. 安装Python环境和依赖库
2. 安装MySQL数据库，创建数据库和用户
3. 导入项目数据和结构
4. 配置Django项目环境
5. 运行Django项目
